Тема:
"Заполнить контент страниц с помощью TravelPayouts API"

Исходная формулировка:
"Необходимо актуализировать справочники, дополнить информацией и наладить нужные связи. При этом должна сохраниться информация и текущие правильные связи.

Необходимо с помощью TravelPayouts API реализовать загрузку следующих данных для страниц направлений:
№ рейса, авиакомпания, вылет, прилет, в пути, цена. С кнопкой «найти билеты», при нажатии на которую можно было перейти на страницу поиска с параметрами данного рейса.

Полное ТЗ во вложении [сюда не закачано]. Сайт на Битриксе."
